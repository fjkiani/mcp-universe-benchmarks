## 🔑 Key Concepts (Prerequisites)

Before diving in, you should understand:

1. **Git & GitHub**
   - Forking repositories
   - Creating branches
   - Making commits
   - Creating PRs

2. **Python Basics**
   - Functions and decorators
   - Async/await
   - JSON handling
   - YAML files

3. **Command Line**
   - Running commands
   - Environment variables
   - Virtual environments

4. **JSON**
   - Syntax rules
   - Object structure
   - Array notation

**Don't know these?** Each guide explains as it goes, but familiarity helps!
